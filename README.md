# ASP.NET Core MVC CRUD — Reference Guide

A personal reference guide for building a full CRUD application with ASP.NET Core MVC, Entity Framework Core, and deploying it to a Linux server with Nginx and HTTPS.

## What this covers

- Project setup and default routing
- DbContext registration and environment variable configuration
- Full CRUD with MVC: controllers, Razor views, tag helpers, model binding
- Deployment with Nginx
- SSL/TLS certificates via Certbot

## How to use

Read `guide.md` top to bottom for a step-by-step walkthrough. Each section includes screenshots and explanations of the key concepts.

## Stack

| Layer      | Technology                    |
|------------|-------------------------------|
| Language   | C#                            |
| Framework  | ASP.NET Core MVC              |
| ORM        | Entity Framework Core         |
| Server     | Nginx (Linux)                 |
| SSL        | Let's Encrypt via Certbot     |
